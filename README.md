# AIO

Проект представляет собой объединение разных проектов но с одним названием. Все так или иначе связаны с ML, проект больше тренировочный, будет охватывать оч много направлений в ИИ.
Сейчас в разработке — **AIO Education** — сервис для бесплатного обучения с использованием ИИ-репетитора.

## Текущее состояние разработки

Проект находится в активной стадии разработки. На данный момент реализованы следующие компоненты:
- Настроена БД, почти прописано первое API
- API для озвучки лекций
- Базовые наброски фронтенд-интерфейса

Для интеграции с сервисом сейчас работаю над локальными нейронками.

## Планы развития

В ближайшей перспективе планируется добавление следующих направлений:
- Робототехника, работа с VLA моделями (Vision-Language-Action) и LeRobot

## Инструкция по запуску

Гайд по развертыванию и запуску проекта будет добавлен после завершения основных компонентов и стабилизации архитектуры.

Также когда доделаю робота выложу его STL исходники, гайд по печати, сборке, настройке и обучении VLA + LeRobot
